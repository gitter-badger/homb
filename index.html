    Homb

    Welcome Homb

    The Homb project is set of bash extensions for applications. It is intended for use with fresh for managing application configurations, including Homb itself. Homb can be installed standalone or with fresh (recommended). Homb leaves all the application configurations to Fresh while Homb extends applications with various completions, helpers, and tools. See the wiki for details on how to use Homb.

    Homb only works with newer versions of bash and also relies on git. Currently we have tested Homb with newer versions of bash on CentOS/Redhat and flavors of Ubuntu.

    Installing Homb with Fresh

    The Homb project works best with Fresh.

    Install fresh

    Install fresh with the following:

    bash -c "`curl -sL get.freshshell.com`"

    Configure Bash (~/.bash_profile)

    Once fresh is downloaded add this to your ~/.bash_profile file (not ~/.bashrc).

    export HOMBASE="${HOME}/.homb"
export FASHO="${HOME}/.fresh/build/shell.sh" && test -f $FASHO && source $FASHO

    Configure (~/.freshrc)

    Add this to the .freshrc file. In this example we are using "~/.homb/" as the setting for the $HOMBASE environment variable in the .bash_profile file.

    #-------------------------------------------------------------------------------
# Homb: Makes setting $HOMBASE for Homb simply $HOME/.homb
#-------------------------------------------------------------------------------
fresh threeiem/homb . --file=~/.homb/

# Homb resource file to load Homb
fresh threeiem/homb rc

    Homb Alone

    The Homb project used to work alone, but now it requires Fresh to be installed.

    Common Problems

    • Not using bash! Homb is for bash only. Sorry, maybe another time.

    • Non-Interactive Bash Check to see if you are running an interative shell. Check with this command...

    shopt login_shell
    • Graphical Does Not Mean Interactive Some terminal programs like konsole do not start bash interactively. For konsole you will need to modify the profile. Here is the command to run bash interactively.
    bash -l
